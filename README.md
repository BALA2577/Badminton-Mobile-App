🏸 Badminton Mobile App

A simple and user-friendly mobile application to track your badminton matches, scores, and performance over time.

🚀 Features
📅 Log match details (date, game type, score)
🏆 Record results (Win/Loss) with dropdown selection
📈 Add notes for areas of improvement
🗓️ Calendar view to quickly visualize match results (e.g., W-5, L-2)
📊 Helps track consistency and performance trends
🎯 Purpose

This app is built for badminton players who want to stay consistent, analyze their games, and improve by tracking their performance in a structured way.

🛠️ Tech Stack
Built using Google AI Studio (vibe-coded 😉)
Mobile-friendly design
Lightweight and easy to use
